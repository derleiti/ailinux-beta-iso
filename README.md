# AILinux

AILinux ist eine KI-gestützte Linux-Distribution auf Basis von Ubuntu 24.04 + KDE Neon,  
entwickelt mit dem Ziel, ein sich selbst konfigurierendes, lernfähiges und freies Betriebssystem zu sein.

## Features
- Vollwertige KDE-Umgebung
- GPT4All & Ollama vorinstalliert (Offline-KI-Tools)
- Eigene Paketquelle (AILinux Mirror)
- Eigener Installer (Calamares, gebrandet)
- Optionaler Cloud-Modus mit Token-Zugang (zukünftig)
- KI-gestützte Fehleranalyse, Auto-Optimierung, Benchmark-Profiling
