🔗 4. Netzwerk und Service-Block innerhalb chroot
Ziel: DNS-Funktion in chroot, Services nicht automatisch starten.

Beispiel: Konfiguration von /etc/resolv.conf und Blockierung via policy-rc.d, damit keine Dienste automatisch starten.

🧰 5. Chroot-Skript erstellen und mounten
Ziel: Pseudo-Dateisysteme mounten (/dev, /proc, etc.), chroot-Skript dort hinein kopieren und ausführbar machen.

🔒 6. Chroot-Umgebung einrichten & Paketinstallation
Phasen im chroot:

System stabilisieren: locale-gen, apt-utils, etc.

Repos & Branding: Externes Setup-Script ausführen.

Pakete installieren:

Basis, Desktop (kde-full, casper, network-manager…), und Standard-Apps.

Optionale Drittanbieter-Pakete wie Chrome oder Wine falls vorhanden.

AILinux-spezifische Pakete mit fallback bei Fehlschlag.

System konfigurieren: Live-User anlegen, Autologin einrichten (plasma-x11.desktop), Desktop-Verknüpfung, Bootloader aktiv, sudoers.

Cleanup im chroot: unnötige Pakete entfernen, Logs löschen.

🔧 7. Nach chroot: Pseudo-Filesystems sauber unmounten
Ziel: Alle gemounteten Verzeichnisse entfernen — mindestens /dev, /proc, /sys, /run.

Funktion: safe_umount() versucht mehrfach, danach ggf. erzwingen mit -f/-l.

📦 8. SquashFS erzeugen
Ziel: komprimierte Root-Dateisystemabbild erzeugen.

Beispiel:

bash
Kopieren
Bearbeiten
sudo mksquashfs AILINUX_BUILD/chroot AILINUX_BUILD/iso/casper/filesystem.squashfs \
  -e boot -noappend -no-recovery -no-fragments -no-duplicates \
  -b 1M -comp zstd -Xcompression-level 22 -processors "$(nproc)"

💿 10. Bootloader einrichten (BIOS & UEFI)

🎯 11. Finale ISO erzeugen
Befehl:

bash
Kopieren
Bearbeiten
sudo xorriso -as mkisofs \
  -r -V "$ISO_LABEL" -o "$ISO_NAME" \
  -J -joliet-long -l \
  -isohybrid-mbr "$MBR_TEMPLATE" \
  -c isolinux/boot.cat -b isolinux/isolinux.bin \
  -no-emul-boot -boot-load-size 4 -boot-info-table \
  -eltorito-alt-boot -e boot/grub/efi.img -no-emul-boot -isohybrid-gpt-basdat \
  AILINUX_BUILD/iso
